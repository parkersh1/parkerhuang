# Parker Huang — Portfolio Site (Starter)

This is a lightweight, responsive portfolio you can deploy to GitHub Pages in minutes.

## Quick start (local)
Open `index.html` in a browser.

## Customize
1. Replace placeholder links:
   - LinkedIn, GitHub
   - Project links (Live demo / GitHub / Write-up)
2. Replace `resume.pdf`:
   - Put your resume PDF in this folder named `resume.pdf`
3. Optional: add a headshot
   - Replace the `PH` avatar in `index.html` with an `<img>` tag.

## Deploy on GitHub Pages
1. Create a repo named `parkerhuang.github.io` (or any name).
2. Upload these files.
3. In GitHub: Settings → Pages
   - Source: `Deploy from a branch`
   - Branch: `main` and folder `/ (root)`
4. Your site will be live at: `https://<username>.github.io/<repo>/` (or username.github.io if that repo name).

## Custom domain (optional)
GitHub Pages supports custom domains in Settings → Pages.
